### 🛠️ What's Inside?
This project uses a modular system to keep things organized:
* **`filter.txt`**: The main controller file.
* **`filter1-clickadu.txt`**: Targets aggressive Clickadu ad networks.
* **`filter2.txt`**: General Vietnamese trash ads and "forgotten" popups.

### 🚩 Report a Missed Ad
If you see an ad that isn't blocked, please [Open an Issue](https://github.com/BlackCatOfficialytb/for-ublock-with-love/issues) with:

2. A screenshot of the ad.
